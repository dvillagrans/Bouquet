# Manual de Usuario

## 1. Objetivo
Guía de uso para comensales y personal operativo del sistema de mesas virtuales, órdenes y pagos.

## 2. Perfiles de usuario
- Comensal.
- Mesero.
- Cocina.
- Administrador.

## 3. Flujo para comensal
1. Escanear el código QR de la mesa.
2. Ingresar a la mesa virtual.
3. Consultar menú y agregar productos.
4. Confirmar pedido.
5. Consultar consumo individual y/o compartido.
6. Realizar pago individual o colaborativo.

## 4. Flujo para mesero
1. Consultar mesas activas.
2. Revisar estado de órdenes por mesa.
3. Registrar apoyos o ajustes operativos permitidos.
4. Verificar estado de cuenta y pagos pendientes.

## 5. Flujo para cocina
1. Visualizar órdenes confirmadas.
2. Cambiar estado de preparación.
3. Marcar órdenes listas.

## 6. Flujo para administración
1. Gestionar productos del menú.
2. Consultar métricas operativas.
3. Revisar historial de órdenes y pagos.

## 7. Recomendaciones de uso
- Confirmar órdenes antes de enviarlas a cocina.
- Verificar montos antes de ejecutar pago.
- Mantener conectividad estable para actualización en tiempo real.

## 8. Mensajes y estados frecuentes
- Pedido pendiente, en preparación, listo.
- Pago pendiente, autorizado, rechazado.

## 9. Preguntas frecuentes
- ¿Puedo pagar productos de otra persona? Sí, si la mesa permite pago colaborativo.
- ¿Qué pasa si se pierde conexión? El sistema debe reintentar sincronización al reconectar.

## 10. Contacto de soporte
Definir canal de soporte técnico del despliegue (correo, mesa de ayuda o responsable interno).
