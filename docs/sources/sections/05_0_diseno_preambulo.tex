\section{Diseño del Sistema}

En este cap\'itulo se presenta la arquitectura del sistema, el dise\~no detallado
de la base de datos (incluyendo el diccionario de datos), las reglas de negocio
operacionales y el dise\~no de las interfaces de usuario.
