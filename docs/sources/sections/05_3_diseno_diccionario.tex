\subsection{Diccionario de Datos}

Este diccionario describe las entidades principales, atributos y dominios lógicos que conforman
el modelo de datos de Bouquet. Por legibilidad y extensión, se documentan los campos
operativos y analíticos críticos de cada entidad. El esquema completo de relaciones se
encuentra detallado en el diagrama de Entidad-Relación global. Las restricciones específicas,
índices y reglas de integridad se documentan en las reglas de negocio y en el esquema
físico de implementación.
