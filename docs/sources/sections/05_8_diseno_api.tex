\subsubsection{Server Actions Implementadas en Next.js}
\label{sec:server_actions}

Las Server Actions se emplean para consultas y mutaciones internas del panel, evitando
exponer rutas HTTP adicionales cuando la operación se consume únicamente desde componentes
de servidor. Las operaciones críticas públicas, como activación de mesa, unión de
comensales o registro de aportaciones, se mantienen como endpoints API debido a que
requieren contratos HTTP explícitos y validaciones específicas.
