\subsubsection{Mapa de Navegación por Rol}

\noindent\textbf{Comensales.} Home $\rightarrow$ Activar/Unirse $\rightarrow$ Menú $\rightarrow$ Canasta $\rightarrow$ Pago.

\noindent\textbf{Mesero.} Login $\rightarrow$ Mesas $\rightarrow$ Detalle de mesa $\rightarrow$ Tomar pedido.

\noindent\textbf{Chef.} Login $\rightarrow$ Tablero Kanban (Cocina).

\subsubsection{Especificación de Componentes Clave}

\noindent\textbf{Tablero Kanban del Chef (/cocina).} Actualización en tiempo
real vía Supabase Realtime suscrito al canal \texttt{ordenes}. Filtros por
sucursal. Cambio de estados PENDIENTE $\rightarrow$ PREPARANDO $\rightarrow$ LISTO.

\noindent\textbf{Componente de Liquidación (/mesa/:id/pagar).} Desglose de
cuentas por comensal y mesa compartida. Integración con gateway de pagos.
Control de concurrencia para evitar doble pago.
